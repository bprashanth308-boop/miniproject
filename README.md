# Java Programs Collection

## Objective

This project contains basic Java programs covering arithmetic operations, loops, arrays, patterns, strings, and simple validation systems.

## Programs Included

### 1. Area and Perimeter of Rectangle

* Accept length and breadth from the user.
* Calculate area using:

  * Area = Length × Breadth
* Calculate perimeter using:

  * Perimeter = 2 × (Length + Breadth)

### 2. Login System

* Store predefined username and password.
* Accept user input.
* Validate credentials.
* Display login success or failure message.

### 3. ATM PIN Validation

* User has only three attempts.
* Verify entered PIN.
* Grant access for correct PIN.
* Block access after three wrong attempts.

### 4. Factorial

* Read an integer value.
* Calculate factorial using a loop.
* Display the result.

### 5. Fibonacci Series

* Generate Fibonacci numbers.
* Print the series up to N terms.
* Use iterative approach.

### 6. Number Pyramid

* Accept number of rows.
* Print pyramid pattern.
* Display increasing numbers in each row.

### 7. Pascal's Triangle

* Generate Pascal's Triangle.
* Display coefficients row by row.
* Use mathematical formula.

### 8. Remove Duplicate Elements

* Traverse the array.
* Identify duplicate values.
* Print only unique elements.

### 9. Merge Two Arrays

* Combine two arrays into a third array.
* Store elements sequentially.
* Display merged array.

### 10. Character Frequency

* Read a string.
* Count occurrences of each character.
* Display frequency of every distinct character.

## Technologies Used

* Java
* Scanner Class
* Arrays
* Loops
* Conditional Statements
* Strings
